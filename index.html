Examples of Work

Some examples of projects I have worked on in the past that highlight some of the skills I picked up.

Semenatic Analysis on Web Scraped Reviews

Through web scraping with BeautifulSoup and Requests I was able to pull reviews from one of my favorite soup restaurants Yelp page. Inside those reviews I extracted the menu items mentioned and used the text from the reviews to develop either positive, negative, or neutral sentiment for the menu items. I documented my journey and process on Medium and all files are located on Github.

Set up Clinical Trials for Tamr

Built a dataset that made it easier to find doctors with clinical trial experience. Used Python for visualizations, cleaning, and analysis and uploaded notebook to Github.

Password Manager Analysis/ Password Generator

Using passwords collected from Google's Password Manager suggestion function to create a dataset containg passwords that had numbers, letters, and special characters. Using python I was able to derive the frequencies of each of these type character for each password as well as their start characters. The randomness of each password string was measured using Shannon entropy just to gauge which passwords would be a bit harder to guess and would also help me train the Pytorch model used for generating new passwords. All the files as well as the datasets created can be found in this repository.

Movie Presentation for Fullstack Academy

A presentation that explores the most successful action and adventure movies released within the past 50+ years based on their budget and return on investment. Merged, cleaned, and analyzed three different datasets from kaggle to centralize information about movies, their budgets, and their ratings using Tableau to create a dashboard and visualize my findings within a Powerpoint Presentation. Uplooaded on my Github.
